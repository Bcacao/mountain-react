# 프로젝트 이름

## 프로젝트 소개
이 프로젝트는 인스타그램과 같은 형식의 사진 기반의 게시글을 포스팅하고, 포스팅한 게시물에 댓글을 달면서 소통할 수 있는 웹사이트를 만들었습니다.

## 주요 기능
- 로그인, 회원가입 기능
- 게시글 포스팅 기능
- 게시글 수정, 삭제 기능
- 댓글 작성, 삭제 기능

## 기술 스택
- React
- Axios (비동기 요청)

## 배포 방법
